# 🎯 학생 숙제 관리 시스템

입체어휘 4000 & 소리훈련 체계적 관리 시스템

## 📁 프로젝트 구조

```
homework-system/
├── index.html                              # 메인 HTML 파일 (새로운 모듈화 버전)
├── homework-system.html                     # 깔끔한 버전
├── modified_homework_system_special.html    # 원본 파일 (레거시)
├── styles.css                              # 모든 스타일 시트
├── js/                                     # JavaScript 모듈들
│   ├── data-manager.js                     # 데이터 관리 (학생, 숙제, 진도)
│   ├── ui-manager.js                       # UI 렌더링 및 이벤트 처리
│   ├── feedback-manager.js                 # 부모님 피드백 생성 및 관리
│   └── utilities.js                        # 유틸리티 함수들 (CSV, 백업 등)
└── README.md                               # 프로젝트 문서
```

## ✨ 주요 개선사항

### 🏗️ 코드 구조 개선
- **관심사 분리**: HTML, CSS, JavaScript 완전 분리
- **모듈화**: 기능별로 JavaScript 파일 분할
- **유지보수성**: 각 모듈의 책임이 명확히 구분됨

### 📦 모듈별 기능

#### 1. DataManager (`data-manager.js`)
- **학생 데이터 관리**: 추가, 수정, 삭제
- **숙제 데이터 관리**: 일별 숙제 정보 저장/로드
- **진도 관리**: 학생별 어휘 진도 추적
- **로컬 저장소**: 브라우저 로컬 저장소 관리
- **백업/복구**: 데이터 내보내기/가져오기

#### 2. UIManager (`ui-manager.js`)
- **학생 카드 렌더링**: 반별/학생별 UI 생성
- **이벤트 처리**: 사용자 상호작용 관리
- **알림 시스템**: 성공/오류 메시지 표시
- **통계 업데이트**: 완료율 등 실시간 통계
- **모달 관리**: 진도 관리, 요약 등 팝업

#### 3. FeedbackManager (`feedback-manager.js`)
- **피드백 생성**: 학습 결과 기반 맞춤 피드백
- **최종 완료**: 일일 학습 완료 처리
- **부모님 전달**: 클립보드 복사 기능
- **성과 평가**: 완성도별 등급 및 권장사항

#### 4. Utilities (`utilities.js`)
- **CSV 내보내기**: 데이터 스프레드시트 변환
- **일괄 작업**: 반별 숙제 일괄 적용
- **시스템 가이드**: 사용법 안내
- **학생 관리**: 학생 추가/수정/삭제 UI

## 🚀 사용법

### 1. 시작하기
- `index.html` 또는 `homework-system.html`을 브라우저로 열기
- 현재 날짜가 자동으로 설정됨
- 학생 목록이 반별로 표시됨

### 2. 일일 숙제 관리
1. **날짜 선택**: 상단에서 숙제 날짜 선택
2. **반 필터**: 특정 반만 표시하고 싶을 때 사용
3. **학생별 입력**: 각 학생 카드에서 숙제 내용 입력
4. **평가 체크**: 어휘합격, 소리훈련합격 등 체크박스 선택
5. **저장**: 각 학생별 저장 또는 자동저장 (30초마다)

### 3. 부모님 피드백 시스템
1. **피드백 생성**: 학생 카드의 "📨 부모님 피드백" 버튼 클릭
2. **내용 확인**: 자동 생성된 상세 피드백 확인
3. **클립보드 복사**: 카카오톡 등으로 전달하기 위해 복사
4. **최종 완료**: 해당 학생의 일일 학습 완료 처리

### 4. 데이터 관리
- **백업**: "💾 데이터 백업" 버튼으로 JSON 파일 다운로드
- **복구**: "📥 데이터 복구" 버튼으로 백업 파일 업로드
- **CSV 내보내기**: 엑셀에서 볼 수 있는 형태로 내보내기

## 📊 특별 반 vs 일반 반

### 특별반 (가니메데, 유로파 A/B, 타이탄 A/B)
- 어휘시험 (Unit 1-10, 1-3차 + 재시험)
- 소리 훈련 (A-F단계 + 다음 진도)
- 원서수업 (자유 입력)
- 퀴즐릿 (활성화 토글 + URL)
- 문법 학습 (자유 입력)

### 일반반 (Halfmoon A/B, Io A/B)
- 어휘 (자유 입력)
- 소리 (자유 입력)
- 독서/원서 (자유 입력)
- 기타 (자유 입력)

## 💾 데이터 저장

- **자동 저장**: 30초마다 로컬 저장소에 자동 저장
- **수동 저장**: Ctrl+S 단축키로 즉시 저장
- **영구 보존**: 브라우저를 닫아도 데이터 유지
- **백업 권장**: 정기적인 백업으로 데이터 손실 방지

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: CSS Grid, Flexbox, 커스텀 속성
- **Vanilla JavaScript**: ES6+ 클래스, 모듈 패턴
- **Local Storage**: 클라이언트 사이드 데이터 저장
- **Responsive Design**: 모바일/태블릿 지원
